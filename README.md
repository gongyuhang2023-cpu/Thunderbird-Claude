# ThunderCraft AI - Thunderbird 邮件 AI 助手

一款原生集成到 Thunderbird 的 AI 邮件写作插件。直接在写信编辑器中润色、翻译、撰写和回复邮件，无需跳转外部窗口。

## 功能特性

- **润色文本** - 选中文本一键润色，措辞更流畅得体
- **正式化 / 口语化** - 快速切换邮件语气风格
- **中英互译** - 选中文本直接翻译替换
- **智能回复** - 阅读邮件时一键生成专业回复
- **按要求回复** - 输入简要要求，AI 生成完整回复
- **撰写新邮件** - 描述需求，AI 帮你写邮件
- **检查语法** - 自动修正拼写和语法错误
- **总结邮件** - 长邮件快速提取要点
- **自定义指令** - 任意 AI 指令处理选中文本
- **右键快捷菜单** - 写信时右键直接调用常用功能

## 安装

### 方式 1：从 .xpi 安装（推荐）
1. 在 [Releases](../../releases) 页面下载 `thundercraft.xpi`
2. Thunderbird → 附加组件 → 齿轮图标 ⚙ → 从文件安装附加组件
3. 选择下载的 `.xpi` 文件

### 方式 2：开发者模式
1. 克隆本仓库
2. Thunderbird → 附加组件 → 齿轮图标 ⚙ → 调试附加组件
3. 点击"加载临时附加组件"→ 选择 `manifest.json`

## 配置

安装后进入扩展设置页面：

| 配置项 | 说明 |
|--------|------|
| API 地址 | 兼容 OpenAI 格式的 API Base URL |
| API Key | 你的 API 密钥 |
| 模型名称 | 如 `claude-opus-4-6`、`claude-sonnet-4-6`、`gpt-5` 等 |
| Temperature | 0-1，越低越精确，越高越有创意 |
| 最大 Token 数 | 生成文本的最大长度 |
| 系统提示词 | 自定义 AI 行为（可选） |

配置完成后点击 **测试连接** 验证是否正常。

## 使用方式

### 写信时
1. 在写信窗口选中文本
2. 点击工具栏 ⚡ 按钮，或右键 → ThunderCraft AI
3. 选择动作（润色、翻译、正式化等）
4. AI 流式生成结果
5. 点击"替换选中"或"插入邮件"

### 阅读邮件时
1. 打开一封邮件
2. 点击工具栏 ⚡ 按钮
3. 选择"智能回复"、"按要求回复"或"总结邮件"
4. AI 生成内容后，点击"创建回复"自动打开回复窗口并填入内容

### 快捷键
- 自定义输入框中 `Ctrl+Enter` 快速发送

## 技术架构

```
popup.js  <-->  background.js  <-->  compose-script.js (写信窗口 DOM)
                     |
               api/claude.js (API 调用, 流式 SSE 解析)
```

- **background.js** - 中心消息枢纽，编排 API 调用和流式转发
- **compose-script.js** - 注入写信窗口，处理文本选区读取和替换
- **popup/** - 弹出面板 UI，三状态切换（动作列表 → 输入 → 流式输出）
- **api/claude.js** - OpenAI 兼容格式 API 客户端，支持流式响应
- **options/** - 设置页面，含连接测试功能

## 隐私与安全

- API Key 仅存储在浏览器本地存储中，**不会**上传到任何第三方服务器
- 邮件内容仅在用户主动触发 AI 功能时发送到配置的 API 端点
- 源码中不包含任何硬编码的密钥或凭证
